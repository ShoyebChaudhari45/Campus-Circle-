📱 Campus Circle — College Community App

A modern Android-based college community platform that enables students and faculty to connect, communicate, and collaborate efficiently within a secure and structured environment.

🚀 Overview

Campus Circle is designed to simplify communication inside a college by providing a centralized platform for notices, events, and messaging.

It ensures seamless interaction between students and faculty with role-based access and organized content delivery.

✨ Features
🔐 Role-Based Authentication
Secure login system for students and faculty
📢 Notice & Announcement System
Faculty can publish department-wise notices
🎉 Event Management
Create and manage college events
💬 Messaging System
Structured communication between users
🏫 Department Segmentation
Content categorized for better clarity
📊 Admin Dashboard
Manage users, notices, and events efficiently
