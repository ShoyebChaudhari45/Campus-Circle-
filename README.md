# 📱 Campus Circle — College Community App  

## 🚀 Overview  
Campus Circle is designed to simplify communication inside a college by providing a centralized platform for notices, events, and messaging.  

It ensures seamless interaction between students and faculty with role-based access and organized content delivery.  

---

## ✨ Features  

- 🔐 Role-Based Authentication — Secure login system for students and faculty  
- 📢 Notice & Announcement System — Faculty can publish department-wise notices  
- 🎉 Event Management — Create and manage college events  
- 💬 Messaging System — Structured communication between users  
- 🏫 Department Segmentation — Content categorized for better clarity  
- 📊 Admin Dashboard — Manage users, notices, and events efficiently  

---

## 🛠️ Tech Stack  

- Language: Java  
- UI: XML (Material Design)  
- Backend: Firebase  
- Database: Firebase Firestore  
- Authentication: Firebase Auth  
- Tools: Android Studio, Git, Postman

## ⚙️ Setup  

1. Clone the repository  
2. Open in Android Studio  
3. Add your `google-services.json`  
4. Enable Firebase Authentication & Firestore  
5. Build and run the app
